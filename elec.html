Brief summary of my random election generator project

This is a C++ program that runs an election that randomly generates 1 of the 3 parties to be the new leading party of the galaxy. This happens through 7 events that have a 50% chance of happening each day for every electorate . Before running the program please input how many electorates you would like that is 1 - 10 inclusive and the number of campaign days you would like for it to go over that's in between 1 - 30 inclusive.
